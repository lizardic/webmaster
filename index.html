HTML Basics

Head

- DOCTYPE tag initializes html document
- html tag encloses html document
- head tag encloses head of html document
- The head contains CSS styles and files that you want your html document to be linked to.
- The link tag is used to link other files, which can be connected by supplying rel, href, and possibly type attributes
- In this example, CSS styles, normalize.css, the fluid grid, and google fonts are linked, in that order.
- Lastly, the head contains the title of the website.

Body

- The body tag encloses the body element
- The body contains the elements that people generally associate with webpages
- These elements are either on the home page (header) or later pages (section)
- Sections can be given class names which can be referred to when styling the section
- I use div class="row" and div class="column" to break my website into rows and columns for easy formatting. You can place any HTML elements that you could place in the header in these rows, columns, or elsewhere in the section to build webpages after your main home page (the header).
- The line after the closing body tag allows you to easily use ionicons by copy pasting the code given on the ionicons website that resembles the form (ion-icon name="name of icon").

Elements in the Body

- Heading elements can be created using h# tags, where # represents a number between 1 and 6
- h1 is generally a main heading, and h6 is generally the sub-most heading
- Paragraphs can be created using an opening and closing p tag.
- Links and buttons that function as links can be created with an a tag and href attribute, where the href attribute contains the link name
- You can also add a target attribute with value "_blank" to make the link open in a new tab
- You can also use a class attribute with the a tag or any other tag to assign an element to a class
- Images are made with an img tag and src attribute, where the src attribute contains the file name
- You can also use an alt attribute to give an alternative description in case the photo cannot be displayed
- You can create a container using the div tag that is referenced through the class name in the class attribute
- These will be used in a similar way to header, section, or class names since it a container holds a group of HTML elements that are similar in some way

Navigation for Header

- A navigation can be created using nav tags
- We use div class="row" because that creates a row to place elements inside
- Inside the row, we placed an unordered list that has list items that are links
- # is simply a placeholder, you should put the link name in the href attribute

Important Note: Using container div with class="row" can create a row. Then, you can use container div again with class="column" to create columns within your row to, for example, split a row of your webpage into 3 columns.

Key Design Guidelines

Typography (Fonts)

Choose fonts that are very neutral and plain, without little fancy extensions coming out of the letters, such as san-serif fonts. Some examples are Open Sans, Lato, Raleway, Monsterrat, and PT Sans. Use very big fonts with decreased font weights for headings and small fonts with increased line height for paragraphs.

Color

Pick one/two main color(s) to use in a way similar to how I used red/pink on this site, otherwise use mostly black and white. It prevents the website from looking like the website on the left. Use color to draw attention to important elements of the website.

Whitespace

Use margins and padding to space out the HTML elements (paragraphs and headings and things), because you do not want your website to be crammed like the website on the right.

CSS Basics

Box Model

In CSS you will style the HTML elements you have already created, and the box model is important for understanding how to style elements. In the center of an HTML element is the content (like text, for example). Then, there is padding between the content and the element's border. Finally, there is a margin consisting of whitespace surrounding the border. The size of these parts can adjusted to your liking for each element.

Structure of CSS Rule

In CSS, a grouping of code is often referred to as a rule. A rule consists of a selector and properties. The selector can be a type of HTML element, a container, or a class. The defining factor of a selector is that it allows the user to reference 1 or more HTML element so that the element(s) can be styled. The properties in a rule are statements that affect the display of the HTML element(s) referenced by the selector.

Inspect-Observing Other Websites

On Google Chrome, you can right-click on any element on any website and click Inspect. After doing this, an interface will pop up with heading "Elements" in one navigation bar and heading "Styles" in another. Study the code under "Elements" for the HTML behind the website and "Styles" for the CSS. You'll notice that as you hover code, certain elements highlight on the webpage, meaning you are viewing the code that created the element. You can get ideas from any website, even mine. This is important for inspiration, as there is no need to figure out how to create the most beautiful website when so many other people have already attempted to and you can learn from them.
